# Calculator Project

This is a simple C# Windows Forms calculator project.  
It lets you do basic arithmetic and also has a scientific mode for more advanced calculations.

## How to Run
### Run the Precompiled EXE
You can run the calculator directly from the precompiled executable here:  
Calculator Final\orignal\bin\Debug\orignal.exe
The calculator includes:
- Menu- That Contain 3 options
      - Scientific calculator
      - Standard calculator
      - Exit calculator
To run it, just go to the folder above and double-click orignal.exe.
Note: This version was built in Debug mode. If you want a Release version, you can open the solution in Visual Studio and rebuild it.
